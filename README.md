كما تم تعريف عناوين URL المناسبة التي تسمح بعرض قائمة بجميع الأطباء أو الوصول إلى معلومات طبيب محدد، بالإضافة إلى إمكانية إضافة وتعديل وحذف سجلات الأطباء.
